## Среда и подход

Все задачи оформлялись как отдельные самостоятельные блоки внутри одного репозитория

Основная рабочая среда:
- **Ubuntu 24.04**

Где это было уместно, решения делались в виде:
- bash скриптов-однострочников
- YAML-манифестов
- `README.md`

Там, где задача была скорее на анализ и рассуждение, основной артефакт - это `README.md`
Там, где задача требовала практической реализации, рядом лежат и файлы реализации
