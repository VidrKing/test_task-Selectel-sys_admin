# Задача 2 - список всех ESTABLISHED TCP-соединений через lsof

## Требование

Нужно с помощью `lsof` вывести список всех TCP-соединений в состоянии `ESTABLISHED`

## Итоговый bash-однострочник

```bash
lsof -a -iTCP -sTCP:ESTABLISHED "$@"
```

## Что делает команда

* `lsof` - вывод открытых файлов и сетевых сокетов
* `-a` - объединение условий через ***И***
* `-iTCP` - оставляет только TCP (Internet sockets)
* `-sTCP:ESTABLISHED` - оставляет только TCP-соединения в состоянии ESTABLISHED
* `"$@"` - опциональная возможность добавить новые опции к команде при вызове скрипта

## Запуск

```bash
chmod +x /путь/до/скрипта/lsof_list_established_tcp.sh
/путь/до/скрипта/lsof_list_established_tcp.sh <опционально добавить доп опции к команде, например, -nP>
```

## Доп. нюансы

1) Тестировалось только на Ubuntu 24.04 (практически новый, важных изменений не было на хосте). Установка зависимостей под эту ОС не требуется
2) Для работы нужен `lsof` в PATH
3) По идее, на некоторых ОС без sudo может некорректно/неполноотрабатывать
