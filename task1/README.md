# Задача 1 - сумма байтов по IP из access.log

## Требования

Нужно через bash-однострочник вывести суммарное количество байтов по каждому IP в формате:

> 1338 bytes for 1.2.3.4
> 
> 100500 bytes for 123.23.1.2

## Реализация

Через awk (в нём обработка и вывод через вложенные утилиты по типу printf), ну и сортировки через sort. Дополнительно настроен

Создан файл `task1/bytes_by_ip.sh`

## Запуск

> chmod +x /путь/до/скрипта/bytes_by_ip.sh
>
> /путь/до/скрипта/bytes_by_ip.sh /путь/до/файла/логов
> 
> ! Если не передать путь до логов, то скрипт будет в рабочей директории (в которой запустили скрипт) искать access.log

## Доп. нюансы

1) Тестировалось только на Ubuntu 24.04 (практически новый, важных изменений не было на хосте). Установка зависимостей под эту ОС не требуется
2) Нет шебанга с указанием интерпретатора. Всё-таки однострочник

> В случае проблем с запуском на неродном хосте для скрипта приложил в папку Артефакты вывод результатов и примеры ошибок
